# 📊 Sky Sport Analysis - SQL

## 📌 Project Description
Analyzed sports data using SQL to extract insights on player performance, team statistics, and match outcomes. Performed data aggregation, filtering, and joins to identify trends such as top players, winning teams, and performance patterns. Generated meaningful insights to support data-driven decisions.

---

## 🎯 Objectives
-To analyze sports data and extract meaningful insights using SQL
-To identify top-performing players and teams based on performance metrics
-To understand match outcomes and winning patterns
-To perform data aggregation and filtering using SQL queries

---

## 🛠️ Tools & Technologies
- SQL (MySQL)  
- MySQL Workbench / DBeaver (Database Tool)  
- GitHub  

---

## 📂 Dataset Description

## 📈 Key Insights
